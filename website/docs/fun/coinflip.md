This command is a game of heads or tails which users can play.

## Trigger

**Type:** `Command`<br />
**Trigger:** `coinflip`

## Usage

- `-coinflip <heads/tails> <bet>` - Flips the coin, betting `bet` on the result being `heads`/`tails`.
