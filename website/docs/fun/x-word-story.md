This command manages an X word story game.

## Trigger

**Type:** `Regex`<br />
**Trigger:** `\A`

:::note Restrictions

Set this command to _only run_ in your x-word-story channel in the channel restrictions.

:::

## Usage

- Just type the correct amount of words and write a story together :)
